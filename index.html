📬 How Daily Alerts Work

Since this is a static website, true automated emails require a free third-party service. Below are the two easiest options — no coding needed.

Recommended Option 1: LinkedIn Job Alerts (Free)
  1. Go to linkedin.com/jobs
  2. Search: "General Manager" "luxury" OR "5-star"
  3. Click Set Alert → choose Daily
  4. Repeat for each target region or brand
LinkedIn Search String
"General Manager" ("Ritz-Carlton" OR "St. Regis" OR "Waldorf Astoria" OR "Park Hyatt" OR "Six Senses" OR "Fairmont" OR "InterContinental" OR "Four Seasons")
Advanced Option 2: Google Alerts (Free, covers brand career pages)
  1. Go to alerts.google.com
  2. Paste each search below → set frequency to Once a day → your email
  3. Create one alert per brand group
